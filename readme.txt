Steps Starting App:

1. Jalankan perintah `composer install` untuk mengunduh package laravel.
2. Jalankan perintah `npm install` untuk mengunduh package node_module.
3. Jalankan perintah `php artisan migrate --seed` untuk migrasi dan seeding database.
4. Jalankan `npm run dev` untuk membangun aset frontend.
5. Jalankan `php artisan serve` untuk menjalankan server lokal.
6. Buka aplikasi web di [http://127.0.0.1:8000](http://127.0.0.1:8000).
